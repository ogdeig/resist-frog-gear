AFFILIATE ARTICLE TEMPLATE (GitHub Pages Ready)
=================================================
HOW TO USE
1) Replace tokens in index.html like {{TOPIC}}, {{META_DESCRIPTION}}, {{INTRO}}, {{SLUG}}.
2) Duplicate the product <section> for each item and update:
   - image: assets/product-N.jpg
   - title: {{P*_TITLE}} etc.
   - badge: {{P*_ARCHETYPE_BADGE}}
   - link: {{P*_AFFILIATE_URL}} (keep rel="nofollow sponsored")
3) The TOC and JSON-LD ItemList are built automatically by script.js.
4) Publish via GitHub Pages (Settings → Pages → main /root).

ASSETS
- hero.jpg (1200x630)
- favicon.png (512x512)
- product-1.jpg ... product-N.jpg
